# Traffic Data Visualizer 

## Overview

Traffic Data Visualizer and Validator is a Python-based application designed to validate user-input dates, process traffic datasets from CSV files, and visualize traffic flow using an interactive graphical interface built with Tkinter. The system combines data validation, analysis, and visualization to help users understand traffic patterns and trends efficiently.

## Key Features

### Date Validation

* Validates user-entered dates in the DD-MM-YYYY format
* Prevents invalid inputs through robust error handling
* Provides clear and user-friendly error messages

### CSV Data Processing

* Reads and processes traffic datasets stored in CSV files
* Extracts relevant traffic information based on the selected date
* Performs data analysis to identify traffic trends and patterns

### Data Visualization

* Generates graphical representations of traffic flow
* Compares traffic data across two junctions
* Presents information in an easy-to-understand visual format

### User Interface

* Developed using Python Tkinter
* Provides an interactive and user-friendly graphical interface
* Simplifies navigation and data exploration

## Technologies Used

* Python
* Tkinter (Graphical User Interface)
* CSV File Handling
* Data Analysis and Visualization

## How It Works

1. The user enters a date in the required format (DD-MM-YYYY).
2. The system validates the date and checks for errors.
3. Traffic data is loaded from the corresponding CSV dataset.
4. Relevant traffic information is extracted and analyzed.
5. The processed data is visualized through graphs and charts.
6. Results are displayed to the user through the graphical interface.
